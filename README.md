# HoloClipy

## Routes

- `/` -> Home
- `/join` -> Join
- `/login` -> Login
- `/search` -> Search

- `user/edit` -> Edit user
- `user/delete` -> Delete user

- `video/watch` -> Watch Video
- `video/edit` -> Edit Video
- `video/delete` -> Delete Video
- `video/comments` -> Comment on a video
- `video/comments/delete` -> Delete A Comment of a Video
